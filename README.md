# ShiftMate Korea

한국 4조3교대·야간근무자를 위한 수면, 운동, 공부, 부업 루틴 자동 생성 웹앱입니다.

배포 주소: https://shiftmate-korea.pages.dev/

## 핵심 기능

- 근무 타입 선택: D, S, G, Off, 직접 입력
- 근무 패턴 선택: 4조3교대, 3조2교대, 2교대, 야간 고정, 직접 입력
- 목표 선택: 회복 우선, 운동 우선, 공부 우선, 부업 우선, 균형 모드
- 루틴 생성: 오늘 루틴, 내일 루틴, 야간근무 후 회복 루틴, Off day 성장 루틴
- 피로도 점수: 근무 타입, 목표, 루틴 종류 기준 0~100점 표시
- 부업 가능 시간: 오늘 확보 가능한 시간을 분 단위로 표시
- 교대근무 인수인계 메모: 메모만 따로 복사 가능
- 공유용 텍스트 복사: 카카오톡과 메모장에 붙여넣기 쉬운 형식
- localStorage 저장
- 백엔드, 로그인, 결제, 광고 없음

## 3차 개선 내용

- 히어로 섹션 문구 강화
- 결과 카드 상단 요약 추가
- 부업 가능 시간 카드 분리
- 교대근무 인수인계 메모 카드 분리
- 인수인계 메모만 복사하는 버튼 추가
- 공유용 텍스트를 짧고 읽기 쉬운 형식으로 정리
- 하단 SEO 콘텐츠와 FAQ 추가
- 휴대폰 홈 화면 추가 안내 섹션 추가
- 주요 버튼 aria-label 보강
- 선택된 옵션에 `선택됨` 텍스트 표시

## 피로도 점수 기준

- 0~39점: 여유 있음
- 40~69점: 균형 필요
- 70~100점: 회복 우선

G 근무와 야간근무 후 회복 루틴은 피로도를 높게 계산합니다. Off day는 회복 우선인지 성장형 목표인지에 따라 다르게 계산합니다.

## 부업 가능 시간 기준

- 15~30분: 아이디어 메모, 블로그 초안, 가벼운 코드 수정
- 30~60분: 글 1개 초안, 기능 1개 수정, 자료 정리
- 60분 이상: 집중 개발, 콘텐츠 제작, 긴 글 작성

피로도 70점 이상이거나 야간근무 후 회복 루틴이면 짧은 메모형 작업을 추천합니다.

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- localStorage
- Cloudflare Pages 정적 export

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 빌드 확인

```bash
npm run build
```

## Cloudflare Pages 배포 방식

이 프로젝트는 서버 기능 없이 정적 파일로 배포합니다.

- Build command: `npm run build`
- Output directory: `out`
- Next.js 설정: `next.config.ts`의 `output: "export"` 유지

## 정적 export 유지 주의사항

- 서버 API Route 추가 금지
- DB, 로그인, 결제 기능 추가 금지
- 배포에 서버 실행이 필요한 기능 추가 금지
- 데이터 저장은 localStorage 기준 유지

## 주의 문구

이 서비스는 의학적 조언이 아닌 생활 루틴 참고용입니다. 수면 부족, 무리한 운동, 과로를 권장하지 않습니다.
