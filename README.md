# ShiftMate Korea

ShiftMate Korea는 4조3교대 전용이 아니라 다양한 한국 교대근무자를 위한 맞춤 루틴 플래너입니다.

배포 주소: https://shiftmate-korea.pages.dev/

## 핵심 기능

- v2 탭 구조: 오늘 루틴, 주간표, 부업 시간, 인수인계, 가이드
- 오늘 요약 대시보드: 선택 요일, 근무, 피로도, 부업 가능 시간, 권장 모드 표시
- 근무 타입 선택: D 주간, S 오후, G 야간, N 야간, Off 휴무, 직접 입력
- 회사별 근무시간 직접 입력: 근무 이름, 출근 시간, 퇴근 시간, 근무 메모
- 주간 근무표 저장: 월요일부터 일요일까지 D/S/G/N/Off/직접 입력 선택
- 오늘 요일 자동 인식: 브라우저 로컬 시간 기준으로 오늘 근무 강조
- 주간 루틴 요약: 야간근무, 휴무, 회복 우선일, 부업 집중 가능일, 예상 부업 가능 시간 표시
- 요일별 루틴 미리보기: 각 요일 카드에서 짧은 루틴 요약 확인
- 근무 패턴 선택: 4조3교대, 3조2교대, 2교대, 주야비, 주주야야비비, 주간 고정, 야간 고정, 직접 입력
- 목표 선택: 회복 우선, 운동 우선, 공부 우선, 부업 우선, 균형 모드
- 루틴 생성: 오늘 루틴, 내일 루틴, 야간근무 후 회복 루틴, Off day 성장 루틴
- 피로도 점수: 실제 근무 시간, 야간 여부, 다음 날 퇴근 여부, 목표를 기준으로 0~100점 표시
- 부업 가능 시간: 오늘 확보 가능한 시간을 분 단위로 표시
- 교대근무 인수인계 메모: 메모만 따로 복사 가능
- 인수인계 탭: 근무 특이사항, 확인 필요 항목, 다음 근무자 전달사항, 개인 컨디션 메모 입력
- 부업 시간 탭: 오늘/이번 주 부업 가능 시간, 피로도별 추천 작업, 피해야 할 작업 표시
- 가이드 탭: FAQ, 홈 화면 추가 안내, 교대근무 루틴 가이드 표시
- 공유용 텍스트 복사: 카카오톡과 메모장에 붙여넣기 쉬운 형식
- localStorage 저장
- 백엔드, 로그인, 결제, 광고 없음

## v2 화면 구조

ShiftMate Korea v2는 긴 단일 화면 대신 탭 구조를 사용합니다.

- 오늘 루틴: 근무 타입, 근무시간, 패턴, 목표, 루틴 종류, 결과 카드
- 주간표: 월~일 근무표 저장, 오늘 요일 강조, 선택 요일 상세
- 부업 시간: 오늘과 이번 주의 부업 가능 시간, 추천 작업 유형
- 인수인계: 교대근무 인수인계 메모 작성과 복사
- 가이드: 교대근무 루틴 설명, FAQ, 홈 화면 추가 안내

상단에는 오늘 요약 대시보드를 배치해 사용자가 들어오자마자 현재 근무, 피로도, 부업 가능 시간, 권장 모드를 볼 수 있게 했습니다.

## v2.1 UX 개선

- 데스크톱 레이아웃 폭을 넓혀 대시보드와 결과 카드 가독성 개선
- 오늘 루틴 탭을 데스크톱 2열 구조로 변경해 설정과 결과를 나란히 확인
- 직접 입력 근무가 아닐 때는 근무시간 조정 UI를 접어두고 버튼으로 열 수 있게 변경
- 오늘 루틴 탭 상단에 `오늘 추천 루틴 요약` 카드와 `오늘 루틴 보기` CTA 추가
- 부업 시간 탭에 `핵심` 라벨 추가
- 부업 시간 탭에서 이번 주 부업 가능 시간을 크게 표시
- 상단 대시보드에 홈 화면 추가와 재방문을 유도하는 문구 추가

## v2.2 리텐션 패치

- 오늘 컨디션 체크 추가: 좋음, 보통, 피곤함, 매우 피곤함
- 한 줄 컨디션 메모 저장
- 오늘 루틴 완료 체크리스트 추가
- 완료율 표시: 예를 들어 3/5 완료, 60%
- 오늘 요약 대시보드에 컨디션, 완료율, 최근 기록 여부 표시
- 최근 7일 루틴 기록 카드 추가
- 최근 기록 초기화 버튼 추가
- 부업 시간 탭에서 컨디션이 피곤함/매우 피곤함이면 더 보수적인 추천 문구 표시

기록은 생활 루틴 참고용이며 의학적 판단이나 치료 안내가 아닙니다.

## 주간 근무표 기능

월요일부터 일요일까지 각 요일의 근무 타입을 저장할 수 있습니다.

- 선택 가능 값: D 주간, S 오후, G 야간, N 야간, Off 휴무, 직접 입력
- 오늘 요일은 브라우저 로컬 시간 기준으로 자동 강조
- 특정 요일의 `이 요일 루틴 보기` 버튼을 누르면 오늘 루틴 결과가 해당 요일 기준으로 바뀜
- 직접 입력 근무는 현재 저장된 사용자 정의 근무 이름, 출근 시간, 퇴근 시간, 메모를 공통 사용
- `근무표 초기화` 버튼으로 주간표를 기본값으로 되돌릴 수 있음

주간표는 서버에 저장하지 않고 이 기기의 localStorage에만 저장됩니다. 공용 PC에서는 사용 후 초기화를 권장합니다.

## localStorage 저장 방식

- 오늘 루틴 입력값: `shiftmate-korea-routine`
- 주간 근무표: `shiftmate-korea-weekly-schedule`
- 직접 입력 근무 템플릿: `shiftmate-korea-custom-shift-template`
- 인수인계 메모: `shiftmate-korea-handover-note`
- 일일 루틴 기록: `shiftmate-korea-daily-records`

로그인, 서버 저장, DB 저장은 사용하지 않습니다.

## 주간 루틴 요약

주간 근무표를 기준으로 아래 항목을 계산합니다.

- 야간근무 횟수
- 휴무일 수
- 회복 우선 권장일 수
- 부업 집중 가능일 수
- 주간 예상 부업 가능 시간 범위

## 지원하는 근무 패턴

- 4조3교대
- 3조2교대
- 2교대
- 주야비
- 주주야야비비
- 주간 고정
- 야간 고정
- 직접 입력

## 직접 입력 근무시간

회사마다 같은 D/S/G/N이라도 시작 시간과 종료 시간이 다를 수 있습니다.
ShiftMate Korea는 근무 이름, 출근 시간, 퇴근 시간, 근무 메모를 직접 입력할 수 있습니다.

퇴근 시간이 출근 시간보다 빠르면 다음 날 퇴근으로 계산하며, 야간근무 판단과 피로도 계산에 반영합니다.

## 피로도 점수 기준

- 0~39점: 여유 있음
- 40~69점: 균형 필요
- 70~100점: 회복 우선

피로도는 특정 패턴명보다 실제 근무 시간, 야간 시간대 포함 여부, 다음 날 퇴근 여부, 휴무 여부, 목표, 루틴 종류를 기준으로 계산합니다.

## 부업 가능 시간 기준

- 야간근무 후: 15~30분 이하의 짧은 메모형 작업 추천
- 주간근무 후: 30~90분 범위에서 작은 결과물 1개 추천
- 휴무일: 1~3시간 가능하되 회복 블록 우선
- 피로도 70점 이상: 긴 집중 작업보다 짧은 메모형 작업 추천

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 빌드 확인

```bash
npm run build
```

## Cloudflare Pages 배포 방식

이 프로젝트는 서버 기능 없이 정적 파일로 배포합니다.

- Build command: `npm run build`
- Output directory: `out`
- Next.js 설정: `next.config.ts`의 `output: "export"` 유지

## 정적 export 유지 주의사항

- 서버 API Route 추가 금지
- DB, 로그인, 결제 기능 추가 금지
- 배포에 서버 실행이 필요한 기능 추가 금지
- 데이터 저장은 localStorage 기준 유지

## 주의 문구

이 서비스는 의학적 조언이 아닌 생활 루틴 참고용입니다. 수면 부족, 무리한 운동, 과로를 권장하지 않습니다.
