# ShiftMate Korea

ShiftMate Korea는 4조3교대 전용이 아니라 다양한 한국 교대근무자를 위한 맞춤 루틴 플래너입니다.

배포 주소: https://shiftmate-korea.pages.dev/

## 핵심 기능

- v3 탭 구조: 오늘 루틴, 월간 캘린더, 주간표, 부업 시간, 인수인계, 가이드
- 월간 캘린더: 날짜별 D/S/G/N/Off/직접 입력 근무 확인과 날짜 기준 루틴 연결
- 반복 패턴 자동 생성: 시작일, 패턴, 30/60/90일 기간을 선택해 근무표 자동 채우기
- 월간 요약: 근무일, 야간근무, 휴무, 회복 우선 권장일, 예상 부업 가능 시간 표시
- 첫 방문 온보딩: 근무표 등록, 오늘 루틴 확인, 매일 컨디션 기록 안내
- 앱 배포 준비: PWA manifest, 개인정보처리방침, 이용약관, 앱처럼 사용하기 안내
- 오늘 요약 대시보드: 선택 요일, 근무, 피로도, 부업 가능 시간, 권장 모드 표시
- 근무 타입 선택: D 주간, S 오후, G 야간, N 야간, Off 휴무, 직접 입력
- 회사별 근무시간 직접 입력: 근무 이름, 출근 시간, 퇴근 시간, 근무 메모
- 주간 근무표 저장: 월요일부터 일요일까지 D/S/G/N/Off/직접 입력 선택
- 오늘 요일 자동 인식: 브라우저 로컬 시간 기준으로 오늘 근무 강조
- 주간 루틴 요약: 야간근무, 휴무, 회복 우선일, 부업 집중 가능일, 예상 부업 가능 시간 표시
- 요일별 루틴 미리보기: 각 요일 카드에서 짧은 루틴 요약 확인
- 근무 패턴 선택: 4조3교대, 3조2교대, 2교대, 주야비, 주주야야비비, 주간 고정, 야간 고정, 직접 입력
- 목표 선택: 회복 우선, 운동 우선, 공부 우선, 부업 우선, 균형 모드
- 루틴 생성: 오늘 루틴, 내일 루틴, 야간근무 후 회복 루틴, Off day 성장 루틴
- 피로도 점수: 실제 근무 시간, 야간 여부, 다음 날 퇴근 여부, 목표를 기준으로 0~100점 표시
- 부업 가능 시간: 오늘 확보 가능한 시간을 분 단위로 표시
- 교대근무 인수인계 메모: 메모만 따로 복사 가능
- 인수인계 탭: 근무 특이사항, 확인 필요 항목, 다음 근무자 전달사항, 개인 컨디션 메모 입력
- 부업 시간 탭: 오늘/이번 주 부업 가능 시간, 피로도별 추천 작업, 피해야 할 작업 표시
- 가이드 탭: FAQ, 홈 화면 추가 안내, 교대근무 루틴 가이드 표시
- 공유용 텍스트 복사: 카카오톡과 메모장에 붙여넣기 쉬운 형식
- localStorage 저장
- 백엔드, 로그인, 결제, 광고 없음

## v2 화면 구조

ShiftMate Korea v2는 긴 단일 화면 대신 탭 구조를 사용합니다.

- 오늘 루틴: 근무 타입, 근무시간, 패턴, 목표, 루틴 종류, 결과 카드
- 주간표: 월~일 근무표 저장, 오늘 요일 강조, 선택 요일 상세
- 부업 시간: 오늘과 이번 주의 부업 가능 시간, 추천 작업 유형
- 인수인계: 교대근무 인수인계 메모 작성과 복사
- 가이드: 교대근무 루틴 설명, FAQ, 홈 화면 추가 안내

상단에는 오늘 요약 대시보드를 배치해 사용자가 들어오자마자 현재 근무, 피로도, 부업 가능 시간, 권장 모드를 볼 수 있게 했습니다.

## v2.1 UX 개선

- 데스크톱 레이아웃 폭을 넓혀 대시보드와 결과 카드 가독성 개선
- 오늘 루틴 탭을 데스크톱 2열 구조로 변경해 설정과 결과를 나란히 확인
- 직접 입력 근무가 아닐 때는 근무시간 조정 UI를 접어두고 버튼으로 열 수 있게 변경
- 오늘 루틴 탭 상단에 `오늘 추천 루틴 요약` 카드와 `오늘 루틴 보기` CTA 추가
- 부업 시간 탭에 `핵심` 라벨 추가
- 부업 시간 탭에서 이번 주 부업 가능 시간을 크게 표시
- 상단 대시보드에 홈 화면 추가와 재방문을 유도하는 문구 추가

## v2.2 리텐션 패치

- 오늘 컨디션 체크 추가: 좋음, 보통, 피곤함, 매우 피곤함
- 한 줄 컨디션 메모 저장
- 오늘 루틴 완료 체크리스트 추가
- 완료율 표시: 예를 들어 3/5 완료, 60%
- 오늘 요약 대시보드에 컨디션, 완료율, 최근 기록 여부 표시
- 최근 7일 루틴 기록 카드 추가
- 최근 기록 초기화 버튼 추가
- 부업 시간 탭에서 컨디션이 피곤함/매우 피곤함이면 더 보수적인 추천 문구 표시

기록은 생활 루틴 참고용이며 의학적 판단이나 치료 안내가 아닙니다.

## v3 월간 캘린더 패치

ShiftMate Korea v3는 개인 교대근무자용 근무표 캘린더와 회복/부업 루틴 코치를 연결합니다.

- 월간 캘린더 탭 추가: 오늘 날짜, 선택 날짜, 날짜별 근무 타입을 한눈에 확인
- 근무 타입별 색상과 텍스트 라벨 표시: D, S, G, N, Off, 직접 입력
- 날짜 클릭 시 상단 대시보드와 루틴 결과가 선택 날짜 기준으로 변경
- 반복 패턴 생성: 4조3교대 기본, 3조2교대 기본, 2교대 기본, 주야비, 주주야야비비, 직접 패턴 입력
- 직접 패턴 예시: `D,D,S,S,G,G,Off,Off`
- 이번 달 근무표 복사와 선택 날짜 루틴 복사 기능 추가
- 주간표는 `이번 주 빠른 편집`, 월간 캘린더는 `장기 근무표 관리` 역할로 분리
- 앱처럼 사용하기 안내와 manifest 파일 추가

복사와 요약 문구는 생활 루틴 참고용이며 수익 보장, 의학적 조언, 과로 권장 표현을 포함하지 않습니다.

## 주간 근무표 기능

월요일부터 일요일까지 각 요일의 근무 타입을 저장할 수 있습니다.

- 선택 가능 값: D 주간, S 오후, G 야간, N 야간, Off 휴무, 직접 입력
- 오늘 요일은 브라우저 로컬 시간 기준으로 자동 강조
- 특정 요일의 `이 요일 루틴 보기` 버튼을 누르면 오늘 루틴 결과가 해당 요일 기준으로 바뀜
- 직접 입력 근무는 현재 저장된 사용자 정의 근무 이름, 출근 시간, 퇴근 시간, 메모를 공통 사용
- `근무표 초기화` 버튼으로 주간표를 기본값으로 되돌릴 수 있음

주간표는 서버에 저장하지 않고 이 기기의 localStorage에만 저장됩니다. 공용 PC에서는 사용 후 초기화를 권장합니다.

## 월간 캘린더 기능

월간 캘린더는 장기 근무표 관리용입니다.

- 현재 월 기준 달력 표시
- 이전 달, 다음 달 이동
- 오늘 날짜와 선택 날짜 텍스트 라벨 표시
- 날짜별 근무 타입 배지 표시
- 선택 날짜 상세 카드에서 피로도, 부업 가능 시간, 추천 루틴 확인
- 반복 패턴으로 30일, 60일, 90일 근무표 생성
- 생성 전 덮어쓰기 확인 문구 표시
- 월간 근무표와 선택 날짜 루틴 텍스트 복사

월간 캘린더 데이터도 서버에 저장하지 않고 이 기기의 localStorage에만 저장됩니다.

## localStorage 저장 방식

- 오늘 루틴 입력값: `shiftmate-korea-routine`
- 주간 근무표: `shiftmate-korea-weekly-schedule`
- 직접 입력 근무 템플릿: `shiftmate-korea-custom-shift-template`
- 인수인계 메모: `shiftmate-korea-handover-note`
- 일일 루틴 기록: `shiftmate-korea-daily-records`
- 월간 근무표: `shiftmate-korea-monthly-schedule`
- 선택 날짜: `shiftmate-korea-selected-date`
- 온보딩 숨김 여부: `shiftmate-korea-onboarding-hidden`

로그인, 서버 저장, DB 저장은 사용하지 않습니다.

월간 근무표를 기준으로 아래 항목을 계산합니다.

- 이번 달 근무일 수

- 회복 우선 권장일 수

- 3조2교대
- 2교대
- 주야비
- 주주야야비비
- 주간 고정
- 야간 고정
- 직접 입력

## 직접 입력 근무시간

회사마다 같은 D/S/G/N이라도 시작 시간과 종료 시간이 다를 수 있습니다.
ShiftMate Korea는 근무 이름, 출근 시간, 퇴근 시간, 근무 메모를 직접 입력할 수 있습니다.

퇴근 시간이 출근 시간보다 빠르면 다음 날 퇴근으로 계산하며, 야간근무 판단과 피로도 계산에 반영합니다.

## 피로도 점수 기준

- 0~39점: 여유 있음
- 40~69점: 균형 필요
- 70~100점: 회복 우선

피로도는 특정 패턴명보다 실제 근무 시간, 야간 시간대 포함 여부, 다음 날 퇴근 여부, 휴무 여부, 목표, 루틴 종류를 기준으로 계산합니다.

## 부업 가능 시간 기준

- 야간근무 후: 15~30분 이하의 짧은 메모형 작업 추천
- 주간근무 후: 30~90분 범위에서 작은 결과물 1개 추천
- 휴무일: 1~3시간 가능하되 회복 블록 우선
- 피로도 70점 이상: 긴 집중 작업보다 짧은 메모형 작업 추천

## Cloudflare Pages 배포 방식

이 프로젝트는 서버 기능 없이 정적 파일로 배포합니다.

- Build command: `npm run build`
- Output directory: `out`
- Next.js 설정: `next.config.ts`의 `output: "export"` 유지

## 정적 export 유지 주의사항

- 서버 API Route 추가 금지
- DB, 로그인, 결제 기능 추가 금지
- 배포에 서버 실행이 필요한 기능 추가 금지
- 데이터 저장은 localStorage 기준 유지

## 앱스토어 배포 전 향후 과제

- 실제 앱 아이콘 제작 및 manifest 아이콘 등록
- 외부 캘린더 내보내기 또는 PDF 저장 기능 검토
- 알림은 서버 없이 가능한 범위에서만 검토
- 수익 보장처럼 보이는 문구 없이 유료 확장 가능성 검토
- Capacitor 또는 TWA 방식의 앱 패키징 검토

## PWA manifest

`public/manifest.json`에 아래 값을 설정했습니다.

- name: `ShiftMate Korea - 교대근무 루틴 플래너`
- short_name: `ShiftMate`
- display: `standalone`
- orientation: `portrait`
- categories: `productivity`, `lifestyle`, `health`
- start_url: `/`
- scope: `/`

현재 실제 아이콘 파일은 추가하지 않았습니다. 존재하지 않는 아이콘 경로를 manifest에 넣지 않아 배포 오류를 피합니다.

## 정책 페이지

- 개인정보처리방침: `/privacy`
- 이용약관: `/terms`

두 문서는 앱 배포 준비용 초안입니다. 실제 운영 전 운영자 정보, 문의 이메일, 법률 검토 결과에 맞게 수정해야 합니다.

## 앱스토어 소개 문구 초안

앱 이름 후보:

- ShiftMate Korea
- ShiftMate Korea - 교대근무 루틴 플래너
- ShiftMate - 교대근무 캘린더

짧은 설명 80자 이내:

교대근무자를 위한 근무표·수면·회복·부업 루틴 플래너

긴 설명:

ShiftMate Korea는 회사마다 다른 교대근무표를 월간 캘린더로 정리하고, 선택한 날짜와 근무 타입에 맞춰 수면, 회복, 운동, 공부, 부업 루틴을 확인할 수 있는 생활 루틴 참고용 앱입니다. 반복 패턴으로 근무표를 빠르게 채우고, 오늘 컨디션과 체크리스트를 기록해 최근 7일 흐름을 볼 수 있습니다. 로그인, 서버 저장, 결제, 광고 없이 이 기기 안에만 데이터를 저장합니다.

주요 기능 5개:

- 월간 근무표 캘린더
- 반복 교대 패턴 자동 생성
- 오늘 루틴 추천
- 부업 가능 시간 계산
- 컨디션 체크와 최근 7일 기록

키워드 후보:

- 교대근무
- 교대근무표
- 4조3교대
- 야간근무
- 수면 루틴
- 회복 루틴
- 부업 시간
- 근무표 캘린더
- 루틴 플래너

스크린샷 제목 후보:

- 월간 근무표를 한눈에
- 오늘 근무에 맞춘 루틴
- 부업 가능 시간 확인
- 컨디션과 체크리스트 기록
- 서버 저장 없는 개인 루틴 앱

## 앱 아이콘 후속 작업

앱 배포 전 아래 아이콘 파일을 제작해야 합니다.

- 512x512 PNG: PWA 및 Android 기본 아이콘
- 192x192 PNG: Android 홈 화면 아이콘
- 180x180 PNG: Apple touch icon
- 1024x1024 PNG: 앱스토어 등록용 원본

아이콘 파일이 준비되면 `public/icons`에 추가하고 manifest의 `icons` 항목과 Apple touch icon metadata를 연결합니다.

## 주의 문구

이 서비스는 의학적 조언이 아닌 생활 루틴 참고용입니다. 수면 부족, 무리한 운동, 과로를 권장하지 않습니다.
